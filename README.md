### Technologies and database that we used

*Backend : Python
*Framework: Django
*Frontend: HTML, CSS, JavaScript
*Database: PostgreSQL
*Deployment: Docker
*Server: Gunicorn
*Proxy-Server: Nginx
*Code Repo: BitBucket
*Hosted: AWS
*Storage: S3 bucket
